General
Base Translate The home translation of the helper bone.
Base Rotate The home rotation of the helper bone.

Triggers
New Trigger Creates a new trigger with the current position/orientation of the helper bone and trigger bone.
Tolerance Adjusts the fade in/fade out of the individual triggers.
Draw Checkbox for drawing or hiding each trigger.
Axis The orientation of the trigger bone.
Rotate The amount of rotation that is added to Base Rotate when this trigger is active.
Translate The amount of translation that is added to Base Translate when this trigger is active.
Snap Snaps the trigger bone to the value stored in Axis. This can be used to quickly view the different triggers.
Upd (Update) Used to adjust the position/rotation of the helper bone. Use Snap, then adjust the position/rotation of the helper, and press Update to adjust the Rotate and Translate values.
Del (Delete) Deletes the trigger.
Copy Makes a new copy of the trigger, which can then be changed and updated with the Update button.
Name Renames the trigger. Useful so they can have more descriptive names, like home, up, down.

Draw Controls
Trigger Draw Axis Allows a change from the default draw axis of the cones and goal orientation.
Max Radius Changes the size of the cones.
Subdivisions Adjusts the density of the wireframe of the cones.
Draw Wireframe Settings for shading the cone.
Wire On Shaded Toggles the wireframe drawing of the cone.
