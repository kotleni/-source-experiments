Step 1. Open the noHelperBones.ma scene.
This scene is already skinned and textured, and has joints that are ready to be made into helper bones.
The first joint that is selected is the trigger joint, and then the joint that will be driven.
Step 2. Select bip_hand, then hlp_forearm.
Choose Valve Tools->Character Rigging->Create Helper Bone
The helper bone node will be a child of hlp_forearm.

The mel command for this would be:
   vstHelperBone -create bip_hand hlp_forearm;
It can also take a name for the helper bone:
   vstHelperBone -create -name hlp_forearmShape bip_hand hlp_forearm;
Step 3. Rotate bip_hand.
In the finished example file, this value is 89 degrees in Y.
Step 4. Rotate hlp_forearm.
In the finished example file, this value is 65 degrees in Y.
Step 5. With the helper bone node selected, open the attribute editor.
Click new trigger.
Step 6. The tolerance for the new trigger needs to be set.
This value sets the cone of influence that affects how the helper bone is driven.
The higher the value, the sooner the helper bone will start moving to achieve its target.
Step 7. Drag the slider to set tolerance - for a forearm twist, 90 is a good value.
Step 8. Create another trigger in -Y.
Triggers can also be created in mel, so for the negative twist on the forearm it would be:
   vstHelperBone -newTrigger -tolerance 90 -axis 0 -89 0 -rotate 0 -65 0 -translate 0 0 0 hlp_forearm;
The arm can now twist in both directions, and the helper bone will be driven.
Step 9. Make a helper bone on bip_lowerArm and hlp_bicep
Step 10. Position bip_lowerArm and hlp_bicep, and create a new trigger.
Step 11. If the tolerance is set to 90 on both triggers, the bicep never achieves the target that was specified.
Step 12. Adjust the tolerance so that the cones of influence allow the helper bone to reach the target.
Step 13. The helper bone will now be driven when the arm rotates.
